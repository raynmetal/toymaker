Detailed Description

template<typename TSystemDerived, typename ... TListenedForComponents, typename ... TRequiredComponents>
class ToyMaker::System< TSystemDerived, std::tuple< TListenedForComponents... >, std::tuple< TRequiredComponents... > >

The base class for any built-in or user-defined system that would like to be hooked to the engine's event cycle or have access to entities and components.

Template Parameters
TSystemDerivedThe derived class to this base class, as in CRTP.
TListenedForComponentsA tuple of components that this system will listen for updates on for its active entities.
TRequiredComponentsA tuple of components that indicate what an entity must have in order to be eligible for this system.
See also
ECSWorld::registerSystem()
