Detailed Description

UI aspect responsible for managing and rendering a NineSlicePanel texture on the UI.

Usage:

Its appearance in JSON is as follows:

{
"type": "UIPanel",
"anchor": [1.0, 0.0],
"content_size": [454.0, 764.0],
"panel_resource_name": "Bad_Panel"
}
