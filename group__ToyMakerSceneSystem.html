◆ TransformInheritMode

enum class ToyMaker::TransformInheritMode : uint8_t
strong

The different kinds of values that can be inherited from a transform belonging to a parent.

Enumerator
PARENT 

Transform components are inherited from this object's parent.

GLOBAL 

This object's transform is directly specified as a world-relative matrix (where the world matrix is simply the identity matrix)

CAMERA 

This object's transform is specified relative to the view matrix of the active camera in its ECS world.
